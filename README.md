### sso-consumer

#### Phía sử dụng dịch vụ authentication

Các bước chạy

1. `cd sso-consumer`
2. Cài đặt thư viện `yarn`
3. Khởi chạy trên localhost `yarn dev`

### sso-provider

#### Phía cung cấp dịch vụ authentication

Các bước chạy:

1. `cd sso-server`
2. Cài đặt thư viện `yarn`
3. Khởi chạy trên localhost `yarn dev`
